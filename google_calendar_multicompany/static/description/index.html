Google Calendar Multi-Company Sync Fix

Synchronisez vos agendas Google sans conflit multi-sociétés

Fini la limitation à un seul projet Google Cloud pour toute votre instance Odoo. Chaque société peut désormais utiliser ses propres Credentials OAuth pour une synchronisation fluide et isolée.

Pourquoi installer ce module ?

Flux Isolé

La Société A ne partage plus les mêmes jetons que la Société B. Étanchéité garantie.

Multi-Domaines

Supporte plusieurs domaines Google Workspace (@societe-a.com, @filiale-b.fr) simultanément.

OAuth Sécurisé

Redirection dynamique vers le bon projet Google Cloud selon la société de l'utilisateur.

Aperçu de la configuration

Le module masque les champs globaux dans les Paramètres Généraux pour éviter toute confusion.

Paramètres spécifiques par Société

Retrouvez la configuration Google Calendar directement dans la fiche de chaque société.

Synchronisation Transparente

Pour l'utilisateur final, le processus reste inchangé : un simple clic sur "Synchroniser" utilise les bons identifiants en arrière-plan.

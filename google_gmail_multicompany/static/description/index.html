Google Gmail Multi-Company OAuth

Utilisez des identifiants OAuth distincts par société

Nativement, Odoo impose un seul Client ID et Client Secret pour toute l'instance. Ce module débloque cette limitation en permettant une configuration isolée pour chaque entreprise.

Fonctionnalités Clés

Configuration par Société

Stockez les credentials Google Cloud directement sur la fiche de chaque société (res.company).

SMTP Dédié

Liez vos serveurs de mails sortants à une société pour utiliser le bon projet OAuth automatiquement.
